Detailed Description

A class to store LC-MS data.

The LC_MS class is used to store LC-MS data. It contains the intensity, mz, ionization mode, measurement mode, MS level, retention time, and scan number of the LC-MS data. The LC_MS class is a subclass of the MassSpectrum class. MassSpectrum

Parameters
intensityThe intensity of the MS signal
mzThe mass-to-charge ratio of the MS signal
ionizationModeThe ionization mode, i.e. whether the signal is from positive or negative mode
measurementModeThe measurement mode, i.e. whether the signal is profile or centroid
msLevelThe MS level, i.e. whether the signal is from MS1 or MS2
retentionTimeThe retention time of the LC-MS signal
scanNumberThe scan number of the LC-MS signal
